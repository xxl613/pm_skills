# LLM Wiki 入口说明

本目录是 `08_LLM_WIKI/` 的入口层，用于帮助 AI 和项目负责人快速理解：

- 当前 Wiki 如何使用。
- 不同任务应该先读哪些资料。
- 哪些内容是稳定口径，哪些只是候选结论。
- 新文件应该如何被消化和登记。

`08_LLM_WIKI/` 不替代原始文件。原始 PRD、调研、技术资料、设计文件、会议纪要和验证材料仍保存在 `00` 到 `07` 目录中。

## 推荐读取顺序

1. `AGENTS.md`
2. `MEMORY.md`
3. `08_LLM_WIKI/00_入口/index.md`
4. 按任务类型读取对应 Wiki 页面。
5. 必要时再打开原始文件。
