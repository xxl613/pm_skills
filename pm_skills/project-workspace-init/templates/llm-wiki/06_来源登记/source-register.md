# 来源登记

本文件登记已经被纳入 LLM Wiki 管理范围的来源文件。

登记不代表文件内容已经成为稳定口径。它只表示该文件已被识别、分类或消化。

来源文件被归档或删除时，必须同步更新对应条目的状态（已归档 / 已删除）并在「影响范围」注明去向（归档后的新路径，或删除原因），保证查询不会引用已不存在的路径。

## 记录格式

```md
| 来源文件 | 类型 | 状态 | 消化记录 | 影响范围 | 最后处理时间 |
|:---|:---|:---|:---|:---|:---|
| path/to/file.md | PRD / 会议纪要 / 调研 / 临时分析 / 原型资料 / 技术资料 | 未消化 / 已摘要 / 有候选结论 / 已确认 / 已归档 / 已删除 | file-digests.md#标题 | 待补充 | YYYY-MM-DD |
```
