# 稳定决策

本文件记录已经确认、后续可直接引用的稳定项目口径。

稳定决策必须由用户明确确认后才能写入本文件。`MEMORY.md`、当前主 PRD、正式交付文件和会议纪要只能作为证据来源，不能单独作为自动写入稳定决策的依据。

## 常见证据来源

- `MEMORY.md` 的当前稳定口径
- 当前主 PRD 或正式交付文件的明确结论
- 用户最新明确确认
- 已确认且未被后续材料覆盖的会议结论

## 记录格式

```md
## 决策标题

- 当前结论：
- 来源：
- 确认方式：
- 影响范围：
- 原候选编号：
- 原问题编号：
- 最后更新：
```
