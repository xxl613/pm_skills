# v5-page-planning

用于把带稳定 ID 的结构化需求参数清单拆成可追溯的页面规划文档集，为绘制大明白 V5 原型建立页面地图、参数覆盖矩阵和逐页低保真规划。

## 适用场景

- 在绘制原型前，把需求或参数清单聚类成页面。
- 明确每页的平台、版式、区域内容和页面跳转关系。
- 用覆盖矩阵检查参数漏接、重复承接和页面断链。
- 将确认后的页面规划交给 `v5-ux-rule` 绘制原型。

## 主要产出

- `_页面地图.md`
- `页面-NN-<slug>.md`
- `页面-NN-<slug>.wireframe.html`
- `页面-NN-<slug>.layout.png`
