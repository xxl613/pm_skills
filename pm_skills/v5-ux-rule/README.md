# v5-ux-rule

用于创建、修改或评审大明白 V5 风格的定制原型，统一桌面端 1440 画布和移动端 375 H5 的页面壳层、设计令牌、组件、对话流与项目建档方式。

## 适用场景

- 按 V5 标准绘制桌面端或移动端定制原型。
- 根据 `v5-page-planning` 产出的页面规划落地高保真页面。
- 复用固定壳层、组件、图标、头像、插画和对话流标准件。
- 维护项目级导航、主题、页面走查和桌面宽度适配。

## 主要内容

- 桌面 A/B/C 与移动 A-m/B-m/C-m 页面壳层。
- 设计令牌、基础组件、成品样例页和真实素材库。
- AI 对话生命周期、多 Agent 协作和结果卡片规范。
- 项目档案、导航同步、桌面适配和走查导航工具。
