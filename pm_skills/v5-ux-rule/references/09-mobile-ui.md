## 顶栏

- 对话页左侧为返回，中心区域是菜单图标 + Agent 名称 + 角色标签，右侧为电话/播报或更多动作。
- 历史、反馈、文件选择等独立页可使用居中标题。
- 40px 触达区域中放 24px 真实 SVG 图标；不使用文字字符临时代替正式图标。

## 页面原型

### A-m 门户

使用移动门户壳层与底部 TabBar。当前全自动课程助教移动画布没有门户真源，因此只沿用既有门户结构；颜色和 Agent 身份继续使用 V5 项目令牌与档案。

### B-m 对话

- 消息流占满顶栏与输入台之间区域。
- 用户消息右对齐；Agent 头像、名称和回复左对齐。
- 任务/文件/结果/追问卡片随内容列宽，左右不得贴边。
- 思考过程可以卡片化展示，但仍处于消息流，不弹桌面抽屉。

### C-m 整页/底部弹层

- 添加资源、模式切换、更多动作、反馈等使用底部弹层。
- 智能体、技能、历史记录等需要搜索与长列表的内容使用整页。
- 弹层背景遮罩 `rgba(0,0,0,.52)`，顶部 16px 圆角，关闭按钮 32px 触达。

## 资产与复用

- 公共 CSS：`assets/components-mobile/mobile-ui.css`。
- 成品页：`assets/examples-mobile/`。
- 图标优先取 `assets/icons/`，头像取 `assets/media/` 或 `assets/avatars/`。
- 新页面优先复制最接近的成品页，只替换业务内容和必要状态。
